sublime的安裝

1.下載好sublime和nodejs後，搜尋cmd<，輸入node -v 檢查nodejs是否有成功安裝

出現v4.4.0,成功 !

2.打開sublime，ctrl+`

在最底下的區塊貼上 packagecontrolSUBLIME TEXT3裏頭的code

按enter等他跑完即可再次輸入ctrl+`把此它收起來

3.ctrl+shift+p輸入PCI

輸入html css js--->使程式碼排列整齊、
AutoFileName --->程式中要貼圖檔或輸入網址的時候，AutoFileName會自動列出選項
emmet --->賦予tab鍵神奇的功能，寫程式時可以省很多打字的時間

至preference->seeting->html/css/js---->node true，完成此動作可以讓程式碼自動排列整齊

4.新增作業區

設定完畢後，接下來就可以開始寫出自己的網頁囉!

sublime快捷鍵

ctrl+S -> 重整

CTRL+S -> 儲存(網頁上才看得到)
